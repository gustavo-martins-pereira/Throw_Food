## 🚀 Description
**Throw Food** is a physics-based interactive project developed using the latest features of **Unity 6**. The project focuses on object manipulation and projectile collider. 

## 🛠️ Technologies
* **Engine:** Unity 6 (6000.0.x)
* **Language:** C#
* **Input System:** Unity New Input System

## ✨ Key Features
* **Physics Interaction:** Accurate projectile collider for "throwing" mechanics.
* **Responsive Input:** Support for cross-platform input handling (Mouse/Keyboard & Touch).

## 📂 Project Structure
```text
├── Assets
│   ├── _Project
│   │   ├── Scripts         # C# Logic and Controllers
│   │   ├── Prefabs         # Reusable Food and Environment objects
│   │   ├── Scenes          # Main gameplay and Test levels
│   │   ├── Materials       # URP Shaders and Textures
│   │   └── Animations      # Throwing and UI animations
├── Packages                # Unity 6 Registry Packages
└── ProjectSettings         # Unity Configuration
```

## ⚙️ How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/gustavo-martins-pereira/Throw_Food.git](https://github.com/gustavo-martins-pereira/Throw_Food.git)
    ```
2.  **Open Unity Hub:**
    * Click on **Add** -> **Add project from disk**.
    * Select the cloned folder.
3.  **Unity Version:**
    * Ensure you have **Unity 6** installed. If not, download it via Unity Hub.
4.  **Play:**
    * Open `Assets/_Project/Scenes/MainScene.unity` and press the **Play** button.
